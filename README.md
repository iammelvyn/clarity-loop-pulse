# LoopPulse
A blockchain-powered health monitoring platform built on Stacks.

## Features
- Record health data with timestamp
- Manage medical professionals access
- Track vital signs and health metrics
- Health data sharing controls
- Emergency access protocol

## Smart Contracts
- `health-records.clar`: Main contract for managing health records
- `access-control.clar`: Manages access permissions for medical professionals
- `emergency-protocol.clar`: Handles emergency access situations
